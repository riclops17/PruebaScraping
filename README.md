# PruebaScraping
Se llevo a cabo una extracción de informacion de la pagina YTS.mx con el uso de la biblioteca de python BeutifulSoup
## información recopilada :
- Titulo de Pelicula
- Director
- Año  
- Genero 
- Rating
- Rating de criticos
- Score de imbd
- Enlaces de torrents
- Sinopsis 
- Cantidad de comentarios
